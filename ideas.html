  • Bynar

    Bynar is an open source system for automating server maintenance across the datacenter. Bynar builds upon many years of experience automating the drudgery of server repair. The goal is to have the datacenter maintain itself.

    Project idea 1. Bynar currently knows how to repair hard drives on Ceph clusters but nothing else. The goal of this project would be to add Gluster drive remediation support to Bynar. This would greatly expand the situations where Bynar can be used.

    Project idea 2. On storage servers you generally have raid controllers installed. While these controllers are helpful in managing storage drives they also create a problem. Tools such as smartmon, libatasmart, etc do not always interact well with raid controllers. The raid controllers expose information about the backing storage drives in their own way with their own command line interface. The goal of this project would be to teach Bynar how to interact with a few major raid controllers thus giving it more accurate information to work under.

    Project idea 3. This is the hardest project. Hard drives expose a variety of information about their status. The goal of this project would be to use artificial intelligence to make accurate predictions about when a hard drive might die. There have been papers written about this that are floating around the internet. Predictions from smart are generally not very accurate but AI predictions are proving very accurate. 80-90% accurate to within a few days of when a drive will die.

    Mentor: Chris Holcombe
